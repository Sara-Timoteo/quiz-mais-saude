# Quiz Mais Saúde — PWA

Versão Progressive Web App (PWA) da app **Quiz Mais Saúde**, reconstruída em HTML/CSS/JS puro. Liga diretamente ao mesmo Supabase do projecto FlutterFlow, instalável em Android e iOS a partir do browser, e pode ser empacotada para a Google Play e App Store sem precisar de Flutter, Android Studio, Xcode ou qualquer outra ferramenta nativa.
